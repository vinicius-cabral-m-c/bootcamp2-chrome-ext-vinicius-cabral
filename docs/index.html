Extensão "Meu Foco" para Google Chrome

Esta é a página do projeto. A extensão foi desenvolvida pelo aluno Vinícius Cabral para ajudar você a se concentrar, limpando a bagunça do seu navegador. Com um clique, feche todas as outras abas e mantenha apenas a tarefa em que está trabalhando.

Você pode ver o código-fonte e baixar a extensão no repositório do GitHub.
